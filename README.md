# pycalc

![version](https://img.shields.io/badge/version-0.0.1-blue)

A simple command-line calculator built with **Python 3.14.0**.

## Features

Here are the features this program currently support.

- Basic arithmetic
- Prime Checker
- Find the Least Common Multiple
- Find the Greatest Common Divisitor
- Find the Remainder
- Factorial
- Permutations
